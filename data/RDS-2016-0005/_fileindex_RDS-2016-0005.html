Urban tree database

Research Data Publication File Index

File Folder Description
_metadata_RDS-2016-0005.html   Metadata file in HTML format containing a description of the content, quality, and other characteristics of the data.
_metadata_RDS-2016-0005.xml   Metadata file in Extensible Markup Language (XML) format containing a description of the content, quality, and other characteristics of the data.
TS1_Regional_information.csv \Data Comma-delimited ASCII text file containing information on the regions for which data were collected; includes city and year of collection.
TS2_Regional_species_and_counts.csv \Data Comma-delimited ASCII text file containing urban tree counts by species for 17 U.S. cities.
TS3_Raw_tree_data.csv \Data Comma-delimited ASCII text file containing information on the raw tree data.
TS4_Growth_eqn_forms.csv \Data Comma-delimited ASCII text file containing the list of equation forms applied to the urban tree data.
TS5_Foliar_biomass_leaf_samples.csv \Data Comma-delimited ASCII text file containing urban foliar data by species for 17 U.S. cities.
TS6_Growth_coefficients.csv \Data Comma delimited ASCII text file containing equations for predicting tree growth parameters by species and predicted parameter in different climate zones.
TS7_Volume_eqns_from_urban.csv \Data Comma-delimited ASCII text file containing volume equations for 28 urban tree species requiring 1) only diameter at breast height (dbh) or 2) dbh and height measurements to calculate volume. (All equations predict aboveground volume in cubic meters per tree.)
TS8_Volume_and_biomass_eqns_from_rural.csv \Data Comma-delimited ASCII text file containing dry weight biomass and volume equations from rural forests.
TS9_Biomass_density_factors.csv \Data Comma-delimited ASCII text file containing an expanded list of biomass density factors for common urban species. Species without a specific equation and/or dry weight density factor are matched to known species.
Errata_Jan2020_PSW-GTR-253.pdf \Supplements Portable Document Format file containing a list of January 2020 changes for the General Technical Report associated with the Urban Tree Database: McPherson, E. Gregory; van Doorn, Natalie S.; Peper, Paula J. 2016. Urban tree database and allometric equations. Gen. Tech. Rep. PSW-GTR-253. Albany, CA: U.S. Department of Agriculture, Forest Service, Pacific Southwest Research Station. 86 p. https://www.fs.usda.gov/treesearch/pubs/52933
Errata_Jan2020_RDS-2016-0005.pdf \Supplements Portable Document Format file containing a list of changes made to Urban Tree Database (https://doi.org/10.2737/RDS-2016-0005) in January 2020.
